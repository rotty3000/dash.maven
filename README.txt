This repository holds the tools for eclipse.org projects using maven.  The tools
are not of general use outside of eclipse, and contain eclipse specific environment
settings.

Projects include:

eclipse-maven-signing-plugin - jar signing for eclipse.org p2 artifacts.
settings.xml - general settings.xml file that eclipse.org projects should use
eclipse-parent - the main parent pom file for use by projects using maven and/or tycho
minerva - a sample eclipse project as well as maven-archetype for contstructing a basic eclipse project with tycho.
